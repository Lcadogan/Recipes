    *Application layers*

*Уровень представления (Presentation Layer): обрабатывает интерфейс, отображаемый пользователю.

*Бизнес-уровень (Business Layer): реализует правила для решения проблемы, для решения которой
    было разработано ваше приложение.

*Уровень сохраняемости (Persistence Layer): содержит логику, связанную с хранилищем базы данных,
    и обрабатывает перевод объектов в форматы базы данных.

*Уровень базы данных (Database Layer): содержит фактическую систему хранения базы данных,
    обрабатывает таблицы, индексы и любые другие операции, связанные с базой данных.
